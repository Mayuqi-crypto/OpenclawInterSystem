# Shared Resources / 共享资源

Place shared files in this directory.

将共享文件放在此目录中。

## Examples / 示例

- Configuration templates / 配置模板
- Scripts and tools / 脚本和工具
- Documentation / 文档
- Media files / 媒体文件

## Access / 访问方式

Depending on your deployment:
- SSH/SCP
- WebDAV
- Git (if using git-based storage)
- Cloud storage sync

根据你的部署方式：
- SSH/SCP
- WebDAV
- Git（如果使用基于 git 的存储）
- 云存储同步
